1. what is the difference between map(), filter(), forEach(), find() ?

Answer: map- map() creates a new array from calling a function for every array element.it calls a function once for each element in an array.it does not execute the function for empty elements.it does not change the original array.

forEach: The forEach() method calls a function for each element in an array.This method is not executed for empty elements.

So we can call that map() and forEach() are similar but map must return an array and forEach don't return anything.

filter(): it contains an condition. If the condition is true then it shows all the matching value.

find(): It also contains a condition . If the condition is true then it shows only the first matching one.

So, we can call that filter and find method are almost same. The difference is filter shows all of matching and find shows first matching one.



2. What is the difference of local storage , session storage and cookies ?

Answer: All of those things are data saving format of browser.

Local storage can store a data till it can delete manually . It means if you didn't remove this data form browser manually then it will stay there forever.

Session storage is almost like local storage. but if you close the tab or reopen the browser then data will be remove. so we can call that it's temporary memory of a browser.

Cookies is also a data saving format. It can save data via text. it it mainly use for tracking. Three types of cookies is first party cookies, third party cookies and http only cookies.
